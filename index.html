Usage Guide

Action Command
Create a students table 
curl -X POST -d '{"key": "id", "fields": {"id": "int", "first_name": "string", "last_name": "string", "grade": "int"}}' [[httpsql-host]]/tables/manage/students
Insert data into the students table 
curl -X POST -d '[{"id": 1, "first_name": "Rick", "last_name": "Sanchez", "grade": 10}, {"id": 2, "first_name": "Morty", "last_name": "Smith", "grade": 9}]' [[httpsql-host]]/tables/students
Get data from the students table 
curl [[httpsql-host]]/tables/students
Delete an entry from the students table 
curl -X DELETE '[[httpsql-host]]/tables/students?field=id&value=2'
Delete the students table 
curl -X DELETE [[httpsql-host]]/tables/manage/students
